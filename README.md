# Snake-Game
This Python project is a graphical user interface (GUI) implementation of the classic snake game. It faithfully replicates the nostalgic experience of controlling a snake and guiding it to eat food and grow longer. The project is developed using Object-Oriented Programming (OOP) principles, making it modular, flexible, and easy to understand.

Features
Gameplay: Players can control the snake using arrow keys or WASD controls to navigate the game board and eat food to grow.
Score Tracking: The project keeps track of the player's score, displaying it on the screen as the snake grows longer.
Collision Detection: The game detects collisions between the snake's head, its body, and the boundaries of the game board, resulting in game over conditions.
GUI Interface: The project utilizes a graphical user interface, providing a visually engaging and interactive experience for players.
OOP Implementation: The code follows Object-Oriented Programming principles, allowing for easy maintenance, extensibility, and code reusability.
